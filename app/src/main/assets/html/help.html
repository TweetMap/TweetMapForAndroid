つぶやきマップとは？

電車の遅延情報がわかるよ！（すごい！）
マップに表示される駅をタップすると遅延情報が表示されます。


使い方

  1. 位置情報の取得
    2. 位置情報の取得をONにすることで、現在地の地図を表示することができます。
    位置情報の取得がOFFになっている場合、アプリ起動時に取得確認画面が表示されます。
    ONにしたい場合は「設定する」を選択し位置情報の取得をONにしてください。
    位置情報の取得をONにしない場合は現在地の表示はできません

  2. Twitter連携
    3. メニュー画面を開いてTwitterにログインしてください。
    Twitterアカウントをお持ちでない場合は、「Twitterアカウントを作成」をタップしてください。
    ※Twitterに認証しない場合、遅延情報の取得ができません。
    ※連携を解除したい場合はTwitterのホームページより連携解除の設定を行ってください

  3. 遅延情報を見る
    4. 遅延情報を確認したい駅を地図上でタップしてください。
    駅及び路線に関する遅延情報が表示されます。
    　
    確認したい駅を検索する場合は「検索」ボタンをタップしてください。
    入力した駅名から候補が表示されますので、見たい駅名をタップすると選択した駅が地図に表示されます。
    　
    タブを切り替える事で検索する遅延情報の時間を指定できます。
